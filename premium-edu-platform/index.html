Pandas Fundamentals

Learn how to manipulate tabular data using Pandas, the most popular data analysis library in Python. We'll cover DataFrames, Series, filtering, and basic transformations.

What is Pandas?

Pandas is a fast, powerful, flexible, and easy-to-use open-source data analysis and manipulation tool built on top of Python. It provides two primary data structures: Series (1-dimensional) and DataFrame (2-dimensional).

pandas_intro.py 
import pandas as pd

# Create a DataFrame from a dictionary
data = {
    'name': ['Alice', 'Bob', 'Charlie'],
    'age': [25, 30, 35],
    'city': ['New York', 'Paris', 'Tokyo']
}

df = pd.DataFrame(data)
print(df.head())

Reading & Writing Data

Pandas supports reading from a wide variety of file formats including CSV, Excel, JSON, SQL, and Parquet. This makes it an essential tool for any data workflow.

io_example.py 
# Read a CSV file
df = pd.read_csv('sales_data.csv')

# Quick overview
print(df.info())
print(df.describe())

# Filter rows
high_value = df[df['revenue'] > 1000]

# Group and aggregate
summary = df.groupby('category')['revenue'].sum()
