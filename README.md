# Swiss Knife for Data Science 🧰📊

SPA-помощник для управления проектами data science.

**Фичи:**
- **Dashboard** — сводка по проектам, задачам, прогрессу
- **Wizard** — пошаговое создание нового DS-проекта
- **Roadmap** — трекер этапов проекта
- **Генераторы** — README, requirements.txt, Dockerfile, GitHub Actions и др.
- **File Inspector** — инспектор файлов проекта
- **CSV Viewer** — просмотр CSV с сортировкой и поиском
- **Code Viewer** — просмотр кода с подсветкой
- **Темы** — dark/light, gamer, cosmic, programmer, retro, techno
- **Локализация** — русский / английский
- **Плагины** — расширяемая архитектура

**Технологии:** Vanilla JS, Chart.js, Font Awesome.

## Быстрый старт

Откройте `index.html` в браузере.
